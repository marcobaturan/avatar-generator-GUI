# avatar-generator-GUI

  A simple GUI for generating simple 2D avatars.

# Instructions

$ python3 virtualenv venv\
$ pip install -r requirements.txt\
$ python main.py\


  Then select the options to create specific avatars.


## Testing

- Ubuntu 22.04
- Laptop HP 
- Hardware:
